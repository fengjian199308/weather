## Installing

```shell
$ composer require fengjian199308/weather -vvv
```

## config

Before using this extension, you need to register an account with the open platform of Golder, and then create an application to obtain the API key of the application.

## Usage

```shell
use Overtrue\Weather\Weather;

$key = 'xxxxxxxxxxxxxxxxxxxxxxxxxxx';

$weather = new Weather($key);
```

## Paramter description

```shell
array|string getWeather(string $city, string $type = 'base', string $format = 'json')
```

- $city - 城市名，比如：“深圳”；
- $type - 返回内容类型：base: 返回实况天气 / all: 返回预报天气；
- $format - 输出的数据格式，默认为 json 格式，当 output 设置为 “xml” 时，输出的为 XML 格式的数据。
